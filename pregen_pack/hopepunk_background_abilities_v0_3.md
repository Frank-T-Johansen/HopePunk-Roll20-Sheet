# Hope//Punk Background Ability Roll20 Pass v0.3
This pass verifies and fills each pregen’s Background Ability into the Roll20 sheet Background Ability slot.

Roll20 sheet slot:
- HTML field name: `attr_background_ability`
- API/import attribute name: `background_ability`
- Value format: `Ability Name — rules text`
