# Hope//Punk Pregen Gear Pass v0.2

This pass adds a structured Civilian Gear catalogue and maps each pregen’s Background gear to mechanical base items. It keeps flavour names from the Background while using the nearest Civilian Gear stat block where possible.

## Summary

- Catalogue items: 225
- Pregens mapped: 20
- Gear rows mapped: 144
- Remaining narrative-only items are intentionally left as inventory notes, not as stat rows.
